# Seed Scripts

## Seed Categories

Create category records in the database:

```bash
pnpm seed:categories
```

This script creates expense categories for organizing costs.

## Seed Source of Income

Create source of income records:

```bash
pnpm seed:source-of-income
```

This script creates subscription plans and one-time income sources.

## Seed BMC Canvas

Seed BMC Canvas sections from JSON:

```bash
pnpm seed:bmc-canvas
```

This script loads BMC Canvas data from `lib/bmc-content.json` into the database.
